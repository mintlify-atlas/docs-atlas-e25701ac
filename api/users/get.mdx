---
title: "Get User by ID"
api: "GET /api/usuarios/{id}"
description: "Retrieve detailed information about a specific user"
---

<Warning>
  This endpoint requires **ADMIN** role or the user must be accessing their own information (authenticated user ID matches the requested ID).
</Warning>

## Authorization

This endpoint requires authentication with a valid JWT token. Access is granted if:
- User has ADMIN role, OR
- User is requesting their own information (ID matches authenticated user)

```bash
Authorization: Bearer <token>
```

## Path Parameters

<ParamField path="id" type="long" required>
  The unique identifier of the user to retrieve
</ParamField>

## Response

Returns a user object with complete information.

<ResponseField name="id" type="long" required>
  Unique identifier for the user
</ResponseField>

<ResponseField name="nombre" type="string" required>
  User's full name (2-50 characters)
</ResponseField>

<ResponseField name="email" type="string" required>
  User's email address (unique in the system)
</ResponseField>

<ResponseField name="rol" type="enum" required>
  User's role in the system. Possible values:
  - `PACIENTE` - Patient
  - `MEDICO` - Doctor
  - `ADMIN` - Administrator
</ResponseField>

<ResponseField name="isActivo" type="boolean" required>
  Whether the user account is active
</ResponseField>

<ResponseField name="fechaRegistro" type="datetime" required>
  Registration timestamp (ISO 8601 format)
</ResponseField>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://api.soraka.com/api/usuarios/1 \
  --header 'Authorization: Bearer <token>'
```

```javascript JavaScript
const userId = 1;
const response = await fetch(`https://api.soraka.com/api/usuarios/${userId}`, {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer <token>'
  }
});

const user = await response.json();
```

```python Python
import requests

user_id = 1
headers = {
    'Authorization': 'Bearer <token>'
}

response = requests.get(f'https://api.soraka.com/api/usuarios/{user_id}', headers=headers)
user = response.json()
```
</RequestExample>
