---
title: "List All Users"
api: "GET /api/usuarios"
description: "Retrieve a complete list of all users in the system with full details"
---

<Warning>
  This endpoint requires **ADMIN** role. Only administrators can access the complete list of users.
</Warning>

## Authorization

This endpoint requires authentication with a valid JWT token and ADMIN role.

```bash
Authorization: Bearer <token>
```

## Response

Returns an array of user objects with complete information.

<ResponseField name="id" type="long" required>
  Unique identifier for the user
</ResponseField>

<ResponseField name="nombre" type="string" required>
  User's full name (2-50 characters)
</ResponseField>

<ResponseField name="email" type="string" required>
  User's email address (unique in the system)
</ResponseField>

<ResponseField name="rol" type="enum" required>
  User's role in the system. Possible values:
  - `PACIENTE` - Patient
  - `MEDICO` - Doctor
  - `ADMIN` - Administrator
</ResponseField>

<ResponseField name="isActivo" type="boolean" required>
  Whether the user account is active
</ResponseField>

<ResponseField name="fechaRegistro" type="datetime" required>
  Registration timestamp (ISO 8601 format)
</ResponseField>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://api.soraka.com/api/usuarios \
  --header 'Authorization: Bearer <token>'
```

```javascript JavaScript
const response = await fetch('https://api.soraka.com/api/usuarios', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer <token>'
  }
});

const users = await response.json();
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer <token>'
}

response = requests.get('https://api.soraka.com/api/usuarios', headers=headers)
users = response.json()
```
</RequestExample>
