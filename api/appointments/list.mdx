title: "List All Appointments"
api: "GET /api/citas"
description: "Retrieve all appointments in the system (Admin only)"
---

## Authorization

This endpoint requires `ADMIN` authority.

<RequestExample>

```bash cURL
curl -X GET https://api.soraka.hospital/api/citas \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.soraka.hospital/api/citas', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const appointments = await response.json();
```

```java Java
HttpRequest request = HttpRequest.newBuilder()
  .uri(URI.create("https://api.soraka.hospital/api/citas"))
  .header("Authorization", "Bearer YOUR_TOKEN")
  .GET()
  .build();
HttpResponse<String> response = client.send(request, HttpResponse.BodyHandlers.ofString());
```

</RequestExample>

## Response Fields

<ResponseField name="id" type="integer" required>
  Unique identifier for the appointment
</ResponseField>

<ResponseField name="pacienteId" type="integer">
  Patient's user ID (null if appointment is available)
</ResponseField>

<ResponseField name="pacienteNombre" type="string">
  Patient's full name (null if appointment is available)
</ResponseField>

<ResponseField name="medicoId" type="integer" required>
  Doctor's ID
</ResponseField>

<ResponseField name="medicoNombre" type="string" required>
  Doctor's full name
</ResponseField>

<ResponseField name="medicoEspecialidad" type="string" required>
  Doctor's medical specialty
</ResponseField>

<ResponseField name="fechaHora" type="string" required>
  Appointment date and time in ISO-8601 format
</ResponseField>

<ResponseField name="estado" type="enum" required>
  Appointment status: `DISPONIBLE`, `CONFIRMADA`, `REALIZADA`, or `CADUCADA`
</ResponseField>

<ResponseField name="motivo" type="string">
  Reason for the appointment (max 255 characters)
</ResponseField>

## Notes

- This endpoint returns all appointments regardless of status
- Only administrators can access this endpoint
- Results include both available slots and booked appointments
